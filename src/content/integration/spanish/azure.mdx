### Azure Security Center [.text-h5]

Envía hallazgos de seguridad directamente a Azure Security Center para una vista unificada de la postura de seguridad de tus recursos Azure.

### Azure Sentinel Integration [.text-h5]

Alimenta Azure Sentinel con eventos de red team IA para detección de amenazas correlacionada y respuesta de seguridad automatizada.

### Logic Apps Automation [.text-h5]

Utiliza Azure Logic Apps para automatizar respuestas a vulnerabilidades críticas, como aislamiento de recursos o aplicación de políticas de seguridad.

### Compliance Monitoring [.text-h5]

Monitorea el cumplimiento de Azure Security Benchmark y otros marcos de cumplimiento utilizando hallazgos de red team IA.
